Infinite scroll quirks

The Infinite Scroll script won't work as expected if there are too few items or the page does not have a scrollbar.

Therefore it is mandatory to add a bit more HTML to the page in order to actually make the user to scroll.

Another option would be to add more items in the pages. This will increase the size of the page and the scrollbar will appear.
